# Bluewud Planner Image Production Library

This folder is the staging area for all externally generated planner imagery used in the Bluewud WoodenBase theme.

Use it in this order:

1. Put parent/source references into `reference-inputs/`
2. Generate outputs into the matching `outputs/` batch folder
3. Move approved finals into `approved-for-theme/`
4. Only after review, copy approved finals into `woodenstreet-theme/assets/`

Related docs:

- [Planner Image Generation Guide](../../docs/planner-image-generation-guide.md)
- [Planner Image Prompt Catalog](../../docs/planner-image-prompt-catalog.md)

Do not put random test renders straight into `woodenstreet-theme/assets/`. Keep the generation pipeline separated until the image is approved.
