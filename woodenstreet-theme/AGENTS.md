# AGENTS.md - woodenstreet-theme

This folder is the canonical Bluewud Shopify theme build.

## Scope

- Work only on storefront theme code here.
- Do not mix Shopify app work from `../woodenbeast/` into this folder.

## High-Risk Files

- `layout/theme.liquid`
- `config/settings_schema.json`
- `assets/section-hydration.js`

## Primary Styling

- Use `assets/wooden-master.css` as the master stylesheet.

## Goal

Build a clean WoodenStreet-style base theme for Bluewud that preserves current live-site coverage and is ready for continued refinement on the dev store.
